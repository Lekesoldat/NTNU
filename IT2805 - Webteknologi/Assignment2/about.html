Questions Assignment 1:

  1. What does the server in the client server architecture do?

    In this specific architecture the server provides functionality and services to the clients requests. When it responds to the requests, it retrieves and transmits files to the client.

  2. What does the client do?

    The clients initiates requests to the server.

  3. What is an IP address?

    An IP-address is a unique label to a device on a network using the TCP/IP-model.

  4. What is DNS?

    Domain Name Server. It works as a phone book, where it maps the symbolic names to their respective IP. The DNS exists so the users does not have to remember every specific IP-adress.

  5. What is the difference between the Internet and WWW?

    The Internet is the entire network of networks. The internet connects the millions of devices together, and the WWW is a way of communicating on the internet. Email, FTP and IRC are other ways. The internet is the networking infrastructure, but the WWW is only a way of accessing the information.

  6. Explain what HTML is

    Hyper Text Markup Language, - a standard markup language for building Web pages. HTML describes the structure of a page.
